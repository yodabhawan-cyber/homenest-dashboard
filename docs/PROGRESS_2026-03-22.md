### 2. **Auto-Merge OpenClaw Config** ✅
**Time:** 30 minutes  
**Status:** COMPLETE

**What we built:**
- Setup wizard now auto-merges agent configs into `~/.openclaw/openclaw.json`
- Removes old HomeNest agents (prevents duplicates)

4. Start using immediately

**1. Dynamic Profile Loading**
- Profiles loaded from JSON instead of hardcoded

**2. Auto-Merge OpenClaw Config**

- Better user experience

**3. Settings Page Architecture**
- Separate from setup wizard
- Tabbed interface for organization
